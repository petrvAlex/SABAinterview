# SABAinterview

**Статический сайт с авторскими материалами по бизнес- и системному анализу.**

---

## О проекте

Этот репозиторий содержит исходники и собранную версию статического сайта, созданного с помощью MkDocs. Проект предназначен как образовательный ресурс и справочник для подготовки к собеседованиям и углублённого изучения тем БА/СА.

Материалы в репозитории представляют собой смесь:
- собственных (авторских) текстов и примеров, созданных автором;
- заимствованных материалов (цитаты, адаптированные выдержки, фрагменты) с указанием источников.
